# Registry parking lot

This folder exists so the repo can become a real shadcn-compatible registry later.

It is intentionally empty right now. Do not market fake registry URLs. Do not write install commands until there are valid registry item JSON files that point to stable component source.

Registry work starts after:

1. Component paths stabilize.
2. Dependencies are decided.
3. Radix-backed components are implemented where necessary.
4. The dogfood site previews the same files that the registry distributes.
