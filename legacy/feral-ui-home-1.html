Four rules. All of them broken differently.

Neobrutalism settled on uniform borders, uniform shadows, uniform radius. Feral keeps the bones and freaks every parameter.

Border width varies

2px to 6px across the same page. The login card is 5px. The alert is 3px. The checkbox is 3px. Nothing agrees and everything coheres.

Shadows go colored

Pink, violet, acid, ultra, tang — the shadow is a second voice, not a photorealistic cast. Double shadows stack two colors.

Radii clash

0px and 999px on the same page. Square pagination next to pill badges next to rounded cards. The mismatch is the identity.

Pattern fills creep in

Stripes, dots, checks, zigzags — applied to surfaces that neobrut left flat. The fill is an accent, not a wallpaper.

Fully customizable set of components.

Frequently asked questions

Are these components accessible?

Fully. Built on shadcn/ui's Radix primitives, every ARIA attribute intact. The visual chaos is a styling layer; the DOM is as clean as the original. Screen readers don't see the rotation or the colored shadows — they see a form.

Why copy/paste and not a dependency?

Same model as shadcn/ui and neobrutalism.dev — the code is yours. Copy it, dial the variance up or down, add your own mutations. Feral is a starting point that refuses to be a ceiling.

Is this just neobrutalism with more colors?

No. Neobrutalism settled: uniform 2px borders, uniform hard shadow, uniform small radius, uniform flat fills. Feral breaks every "uniform." The borders vary. The shadows are colored. The radii clash. Pattern fills appear. Components rotate. The variance across the page IS the aesthetic — not the individual choices.
