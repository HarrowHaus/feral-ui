# Feral UI Phase Blueprint

## Product basis

Feral UI is a controlled-variance React component library and preview site. The public product should include:

- routed docs
- component pages
- blocks
- templates
- Style Lab
- Ornament Pack
- registry receipts
- accessibility notes

## Build order

1. Real routed docs site.
2. Component docs generator and page data.
3. Pages for all UI records.
4. DataTable hardening.
5. Calendar and DatePicker hardening.
6. Toast provider and hook.
7. Carousel controls.
8. Sidebar mobile/collapsible behavior.
9. Ornament Pack expansion.
10. Style Lab full theme generator.
11. Eight template previews.
12. Registry fresh-app proof.
13. Accessibility receipts.
14. Public release files.
15. Competitive reassessment.

## Voice rule

Humor is part of the product. Fake proof is not. Every claim needs a code receipt.
