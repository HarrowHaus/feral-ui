## Acceptance standard

A component is done only when it has:

1. Source file.
2. Preview example.
3. Keyboard behavior validated.
4. Accessible labels/roles where applicable.
5. Style-axis notes.
6. Copy-paste usage snippet.
7. Known limitations listed.
