# feral/ui v0.6 build log

v0.6 executes the first official competitive phases rather than adding more homepage confetti.

## Added

- Hash-routed product/docs shell.
- Direct routes for docs, components, blocks, templates, Style Lab, ornaments, accessibility, and changelog.
- Component catalog generation expanded to every `src/components/ui/*.tsx` source item.
- Component routes for every registry UI record.
- DataTable hardening: search, sorting, pagination, row selection, loading rows, empty state, dense mode, mobile hide hooks.
- Style Lab upgraded into a fuller theme generator:
  - 11 presets.
  - ink/paper/signal color controls.
  - border, pressure, radius, tilt, pattern, density, motion controls.
  - CSS output.
  - Tailwind token output.
  - registry-style JSON preview.
  - downloadable CSS theme file.
- Ornament Pack expanded to 40+ named exports.
- Ornament route with live samples and copyable usage.
- Eight template routes:
  - SaaS landing
  - Dashboard
  - Docs
  - Blog/magazine
  - Ecommerce
  - Agency/portfolio
  - AI chat
  - Changelog
- Accessibility receipts route.

## Still honest

- The registry is still local until hosted fresh-app shadcn install tests exist.
- Some components are still basis-level and marked that way.
- The docs can joke. The install story cannot.
